Installation
==============

### CocoaPods

1. Add `pod "YKLineChartView"` to your Podfile.
2. Run `pod install` or `pod update`.
3. Import \<YKLineChart.h\>.


### Manually

1. Download all the files in the YKLineChartView subdirectory.
2. Add the source files to your Xcode project.
3. Import `YKLineChart.h`.
